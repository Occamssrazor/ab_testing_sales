# A/B Test Analysis Report

## Описание эксперимента

В период с **23.03.2022 по 30.03.2022** был проведён A/B-эксперимент.  
Пользователи были случайным образом разделены на две группы:

- **Control** (`pilot = 0`) — 11 769 пользователей  
- **Pilot** (`pilot = 1`) — 11 564 пользователя  

Цель эксперимента — оценить влияние изменений на бизнес-метрики:
- общую выручку,
- выручку пользователей сайта,
- число заказов,
- среднее время загрузки страниц.

---

## Используемые метрики

### Основные бизнес-метрики
- **total_revenue** — суммарная выручка на пользователя
- **total_revenue_web** — выручка пользователей, заходивших на сайт
- **orders_cnt** — число заказов на пользователя

### Контрольная метрика
- **avg_load_time** — среднее время загрузки страницы (мс)

---

## Статистические методы

Для повышения надёжности выводов использовались несколько подходов:

1. **t-test** (Welch)
2. **Mann–Whitney U test** — непараметрическая проверка устойчивости результатов
3. **CUPED** — снижение дисперсии с использованием ковариаты
4. **Power / A/A анализ** — проверка корректности методологии
5. **Decomposition** — разложение эффекта выручки на бизнес-факторы

Уровень значимости: **α = 0.05**

---

## Результаты t-test

| Метрика | p-value |
|-------|---------|
| total_revenue | 6.5e-15 |
| total_revenue_web | 6.5e-15 |
| orders_cnt | 7.7e-27 |
| avg_load_time | 0.154 |

**Выводы:**
- Эксперимент оказал статистически значимое влияние на все бизнес-метрики.
- Контрольная метрика (время загрузки) статистически не изменилась.

---

## Проверка устойчивости (Mann–Whitney)

| Метрика | MW p-value |
|------|------------|
| total_revenue | 3.7e-18 |
| total_revenue_web | 3.7e-18 |
| orders_cnt | 1.6e-27 |

**Вывод:**
Результаты полностью согласуются с t-test, что подтверждает устойчивость эффекта и отсутствие влияния выбросов.

---

## CUPED-анализ

В качестве ковариаты использовалась историческая выручка пользователя:

- **cov_7d** — за 7 дней до эксперимента
- **cov_28d** — за 28 дней до эксперимента

### Диагностика CUPED

| Ковариата | Корреляция | Снижение дисперсии |
|---------|------------|--------------------|
| cov_7d | 0.05 | ~0% |
| cov_28d | 0.46 | **~21%** |

**Выводы:**
- CUPED с ковариатой за 7 дней не даёт полезного эффекта.
- CUPED с ковариатой за 28 дней существенно снижает дисперсию и повышает мощность теста.
- В текущем эксперименте эффект и без того очень сильный, однако CUPED(28d) рекомендуется как стандартный подход для будущих экспериментов с более слабыми эффектами.

## Decomposition эффекта выручки

Выручка была разложена по формуле:

Revenue = Orders per User × Average Order Value


### Строгая декомпозиция (с interaction)

| Компонент | Доля |
|---------|------|
| Orders effect | **44%** |
| AOV effect | **51%** |
| Interaction | **5%** |

### Проверка AOV
- t-test p-value: 2.2e-15
- Mann–Whitney p-value: 1.8e-18

**Выводы:**
- Рост выручки обусловлен **как увеличением числа заказов, так и ростом среднего чека**.
- Оба эффекта статистически значимы.
- Положительный interaction указывает на то, что эффекты усиливают друг друга.

---

## Итоговые выводы

- Эксперимент показал **статистически значимый и устойчивый рост бизнес-метрик**.
- Эффект подтверждён:
  - параметрическими и непараметрическими тестами,
  - CUPED-анализом,
  - декомпозицией на бизнес-факторы.
- Рост выручки носит **системный характер** и не является следствием отдельных выбросов.
- Эксперимент влияет одновременно:
  - на частоту покупок,
  - на размер заказа.

---

## Рекомендации

- Эксперимент можно **смело рекомендовать к масштабированию**.
- Для будущих экспериментов по выручке:
  - использовать **CUPED с ковариатой за 28 дней**,
  - дополнительно проводить **Mann–Whitney** как robustness check.
- В продуктовой аналитике учитывать decomposition, так как эффект достигается сразу по двум ключевым рычагам.
